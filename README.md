# 🚀 TrustInvoice — Decentralized Invoice Financing Platform

TrustInvoice is a **Blockchain-based Invoice Financing Platform** that allows businesses to **convert invoices into NFTs** and receive **instant funding** from investors.

This platform solves **cash flow problems for MSMEs** by enabling **invoice auctions and decentralized funding**.

---

# 🌟 Features

## ✅ Core Features

- Invoice NFT Minting  
- Auction-based Invoice Funding  
- Investor Bidding System  
- Instant Funding for Businesses  
- Wallet Integration (MetaMask)  
- Smart Contract Settlement  
- Decentralized Marketplace  

---

# 🧠 How It Works

1. **Seller creates invoice**
2. **Invoice minted as NFT**
3. **Seller lists invoice for auction**
4. **Investors place bids**
5. **Seller accepts highest bid**
6. **Seller gets instant funding**
7. **Buyer pays on due date**
